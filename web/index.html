Welcome to Drekkar

Drekkar is a game engine for games that runs on the web.

There are 2 clients, one for 2d games and one for 3d games.

The 3d client works best with Firefox. Chrome works but runs slower. With Safari the mouse input is not working. IE is not tested at all.
